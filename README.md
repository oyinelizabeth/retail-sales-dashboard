# Retail Sales Dashboard Project

## 📊 Overview

This project analyses retail sales data using Tableau to identify key business trends, customer behaviour, and product performance. The goal is to transform raw transactional data into actionable business insights.

## 🛠 Tools Used

* Tableau Public
* Microsoft Excel / CSV dataset

## 📁 Dataset

The dataset contains retail transaction data including:

* Order details
* Customer information
* Product categories
* Sales and regional data

## 📈 Key KPIs

* Total Sales
* Number of Orders
* Number of Customers
* Average Order Value

## 📊 Key Insights

* Identified top-performing regions based on total sales
* Analysed monthly sales trends to understand seasonality
* Found best-selling product categories and top products
* Evaluated customer segmentation performance

## 📌 Dashboard Features

* Interactive KPI dashboard
* Regional performance analysis
* Time-based sales trends
* Product category breakdown
* Top product rankings
* Filterable views for deeper analysis
