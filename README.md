# 📊 Retail Sales Dashboard Project

## 📌 Overview

This project analyses retail sales data using **Tableau Public** to identify key business trends, customer behaviour, and product performance. The aim is to transform raw transactional data into actionable business insights that support decision-making.

---

## 🛠 Tools Used

* Tableau Public
* Microsoft Excel / CSV dataset

---

## 📁 Dataset Description

The dataset contains retail transaction data including:

* Order details (Order ID, Order Date, Ship Date)
* Customer information (Customer ID, Segment, Location)
* Product details (Category, Sub-Category, Product Name)
* Sales and revenue data

---

## 📊 Key KPIs

* Total Sales
* Number of Orders
* Number of Customers
* Average Order Value

---

## 🔍 Key Insights

* Certain regions significantly outperform others in total sales
* Sales trends show seasonal variation across the year
* A small number of products contribute disproportionately to revenue
* Customer segments exhibit different purchasing behaviours
* Product categories vary significantly in performance

---

## 📌 Dashboard Features

* Interactive KPI summary dashboard
* Regional performance analysis
* Time-series sales trend analysis
* Product category and sub-category breakdown
* Top-performing products ranking
* Interactive filters for dynamic exploration
